# 🇪🇺 EU Net Salary Calculator

**Free multi-country take-home pay comparison tool for 20 European countries.**

🔗 **Live Site:** [your-username.github.io/eu-salary-calculator](https://your-username.github.io/eu-salary-calculator)

## Features
- 20 countries: Germany, Netherlands, France, Spain, Italy, Poland + 14 more
- Live EUR/USD/GBP/CHF exchange rates via Frankfurter API (European Central Bank)
- Employee vs Self-Employed modes
- Annual / Monthly toggle
- Save & load comparisons (localStorage — no server needed)
- Shareable URLs
- Fully mobile responsive
- Zero backend — deploys free on GitHub Pages

## Tech Stack
```
HTML5 + CSS3 + Vanilla JavaScript
Frankfurter API — live exchange rates (free, ECB-sourced)
localStorage — save comparisons
GitHub Pages — free hosting
```
